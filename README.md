* OS Fundamentals
* Source Control
* Configuring Development Environments
* Cloud Architecture & Hosting Models
* Cloud Networking
* Serverless
* Containerisation
* Databases
* Storage
* Security
* Infrastructure as Code
* CI/CD
* Cost Control

This shall be demonstrated via the deployment of an Nginx server to EC2
